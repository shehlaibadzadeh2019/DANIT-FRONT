Задача

        сумма кредита,
        количество месяцев
        процентная ставка
        ответ в виде 3х alerts
        1) сколько заплатит заемщик всего
        2) стоимость пользования кредитом
        3) платеж в месяц

План работ

        1.) написать план
        2.) начать реализовывать с тех моментов, которые понятны
        3.) когда работает как сказано, сделать проверку - работает ли код во всех возможнных ситуациях
        4.) сделать функции универсальными
        5.) Оптимизировать код
        6.)

Работа

        1.) получить входные данные о сумме кредита
            1.1.) проверка по null- если null - повторный запрос
            1.2.) проверка число - повторный вопрос, если не число
            1.3.) проверка на отрицательное число
            1.4.) конверт данных в число
    
        2.) получить входные данные по количеству месяцев:
            2.1.) проверка по null- если null - повторный запрос
            2.2.) проверка число - повторный вопрос, если не число
            2.3.) проверка на отрицательное число
            2.4.) проверка диапазомон 0-12
            2.5.) конверт данных в число
            
        3.) получить входные данные по процентной ставке
            3.1.) проверка по null- если null - повторный запрос
            3.2.) проверка число - повторный вопрос, если не число
            3.3.) проверка на отрицательное число
            3.4.) проверка диапазомон 1..360
            3.5.) конверт данных в число
        
        4.) Обработка данных
            4.1) посчитать сумму общего платежа (формула)
            4.2) посчитать стоимость пользования платежом
            4.3) посчить плат. в месяц (формула)
        
        5.) Вывести полученные данные
